About

The N-Gage SDK is a collection of tools, libraries, and documentation that allows developers to create homebrew software for the Nokia N-Gage. It includes a C/C++ toolchain, a collection of example applications, and a set of libraries that are compatible with the N-Gage's hardware.

The following notable libraries are included in the SDK:

  1. Simple DirectMedia Layer (SDL3)
  2. SDL_mixer
  3. NRenderer, VibraControl (Nokia)
  4. Vibra Control (Nokia)
  5. EGL 1.3, OpenGL ES 1.1, OpenVG 1.0 (Hybrid)
